## Install
```bash
$ npm install -g save-me
```

## Usage
```bash
$ save-me --help

  Usage:
    $ save-me          Save an item
    $ save-me <item>   Search and get an item

  Example:
    $ save-me
    $ save-me <item>
    $ save-me -l
    $ save-me -r <item>

  Options:
    -r, --remove       Remove item selected
    -l, --list         List all saved items

    -h, --help         Show help options
    -v, --version      Show version
```
